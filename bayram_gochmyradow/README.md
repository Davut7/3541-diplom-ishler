## Описание проекта

### Что такое Android Security Analyzer?

Android Security Analyzer — это веб-приложение, которое позволяет анализировать APK файлы Android-приложений на предмет:

- **Опасных разрешений** — определение разрешений, которые могут представлять угрозу безопасности
- **Уязвимостей манифеста** — проверка AndroidManifest.xml на небезопасные настройки
- **Оценка рисков** — расчет общего уровня риска приложения
- **Компонентов приложения** — анализ Activities, Services, Receivers, Providers

### Почему стоит использовать этот инструмент?

| Преимущество | Описание |
|--------------|----------|
| **Приватность** | Все анализы выполняются локально, файлы не отправляются на сторонние серверы |
| **Простота** | Интуитивно понятный интерфейс drag-and-drop |
| **Многоязычность** | Поддержка английского и туркменского языков |
| **Детальные отчеты** | Подробные отчеты с рекомендациями по безопасности |
| **Открытый код** | Полностью открытый исходный код |

## Функциональность

### 1. Анализ разрешений (Permissions)

Система анализирует более **70 известных Android разрешений** и классифицирует их по уровню риска:

- 🔴 **Critical (Критический)** — Разрешения, которые могут серьезно угрожать безопасности
  - Камера, микрофон, GPS
  - SMS, звонки
  - Установка приложений

- 🟠 **High (Высокий)** — Разрешения с повышенным риском
  - Контакты, календарь
  - Хранилище
  - Состояние телефона

- 🟡 **Medium (Средний)** — Умеренный риск
  - Интернет
  - Bluetooth
  - Автозагрузка

- 🟢 **Low (Низкий)** — Минимальный риск
  - Вибрация
  - Wake Lock
  - Уведомления

### 2. Анализ манифеста

Проверка AndroidManifest.xml на:

- **Debuggable флаг** — приложение в режиме отладки
- **AllowBackup** — разрешено резервное копирование
- **Exported components** — экспортированные компоненты
- **Min/Target SDK** — версии SDK

### 3. Оценка безопасности

- Общий Risk Score (0-100%)
- Список проблем безопасности
- Предупреждения
- Рекомендации по устранению

### 4. Генерация отчетов

- Экспорт в JSON формат
- Экспорт в HTML формат (красивый отчет для печати)
- Печать отчетов
- Детальная информация по каждому разрешению

### 5. История анализов

- Автоматическое сохранение всех анализов
- Статистика по всем проверенным приложениям
- Графики распределения рисков
- Возможность просмотра и удаления записей

### 6. Дополнительные функции

- Темная/светлая тема интерфейса
- Адаптивный дизайн для мобильных устройств
- Сравнение с другими инструментами безопасности
- Демо-режим для тестирования без загрузки APK

## Как это работает

### Процесс анализа APK

```
┌──────────────┐     ┌──────────────┐     ┌──────────────┐     ┌──────────────┐
│   1. Upload  │────▶│  2. Extract  │────▶│  3. Analyze  │────▶│  4. Report   │
│    APK File  │     │   Manifest   │     │  Permissions │     │   Generate   │
└──────────────┘     └──────────────┘     └──────────────┘     └──────────────┘
       │                    │                    │                    │
       ▼                    ▼                    ▼                    ▼
  Пользователь        APK файл              Проверка             Детальный
  загружает APK      распаковывается,      каждого              отчет с
  через drag &       извлекается          разрешения на        рекомендациями
  drop интерфейс     AndroidManifest      уровень риска
```

### Алгоритм оценки риска

```javascript
Risk Score = Σ (Permission Weight × Risk Level)

Risk Levels:
- Critical: 30 points
- High: 15 points
- Medium: 5 points
- Low: 1 point

Final Score = (Total Points / Max Possible) × 100%
```

## API документация

### Endpoints

#### POST /api/analyze/upload

Загрузка и анализ APK файла.

**Request:**
```
Content-Type: multipart/form-data
Body: apk (file)
```

**Response:**
```json
{
  "success": true,
  "reportId": "uuid-string",
  "analysis": {
    "basicInfo": {...},
    "manifest": {...},
    "permissions": {...},
    "security": {...}
  }
}
```

#### GET /api/analyze/demo

Получение демо-анализа для тестирования.

#### GET /api/analyze/report/:id

Получение сохраненного отчета по ID.

#### GET /api/analyze/permissions

Получение базы данных всех известных разрешений.

#### GET /api/health

Проверка состояния сервера.
